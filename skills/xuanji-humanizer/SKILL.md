# 织梦人化 · ZhiMeng Humanizer

> 让AI写作像人写，璇玑史的文字更有温度

---

## 核心理念

```
AI写作特征:
❌ "首先" "其次" "最后"
❌ "值得注意的是"
❌ "然而" 过度使用
❌ 过于完美的对仗
❌ 缺乏个人风格

人类写作特征:
✅ 自然流畅的过渡
✅ 多样化的句式
✅ 个人情感色彩
✅ 偶有不完美
✅ 独特的表达习惯
```

---

## AI写作常见信号

| 信号类型 | 典型特征 | 消除方法 |
|---------|---------|---------|
| **过渡词** | 首先/其次/最后 | 删除或替换 |
| **填充词** | 值得注意的是/可以说 | 直接表达 |
| **完美对仗** | 结构过于工整 | 增加变化 |
| **笼统表达** | "创新性的解决方案" | 具体化 |
| **情感缺失** | 缺乏温度 | 增加人情味 |

---

## 人性化规则

### 1. 过渡词处理

```
❌ 首先，我们需要...
   其次，...
   最后，...

✅ 先说个背景...
   接下来...
   总结一下...
```

### 2. 句式变化

```
❌ AI喜欢用"这是一个..."
   "从...角度来看..."

✅ 直接说事实
✅ 用短句增强节奏
✅ 适当用问句
```

### 3. 情感注入

```
❌ "该项目取得了显著成效"
✅ "这个项目让我们很兴奋，效果确实不错"
```

### 4. 具体化

```
❌ "提供创新性解决方案"
✅ "让用户省了50%的时间"
```

## 质量评分

```python
def human_score(text):
    score = 100
    
    # 扣分项
    if has_ai_transitions(text): score -= 20
    if too_perfect_structure(text): score -= 15
    if lacks_specificity(text): score -= 10
    if no_emotion(text): score -= 15
    
    # 加分项
    if has_personality(text): score += 10
    if natural_rhythm(text): score += 10
    
    return max(0, score)
```
