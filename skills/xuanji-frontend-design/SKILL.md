# 璇玑史前端设计系统

> 拒绝"村姑审美"，打造顶级视觉体验

---

## 核心理念

| Anthropics设计原则 | 璇玑史执行 |
|-------------------|-----------|
| **独特字体** | ❌ 不用 Arial/Inter | ✅ 特色中英字体 |
| **大胆配色** | ❌ 不用紫白渐变 | ✅ 文化配色体系 |
| **精致动效** | ⚠️ 避免生硬 | ✅ 丝绸般的动画 |
| **空间构图** | ❌ 不用居中外卖 | ✅ 璇玑美学 |

---

## 设计禁区

```
❌ Inter/Roboto/Arial 字体
❌ 紫色渐变 + 白底
❌ 居中对称布局
❌ 圆角卡片堆砌
❌ 平平无奇的hover效果
❌ 通用AI美学
```

## 璇玑美学体系

### 🎨 配色方案

| 主题 | 主色 | 辅色 | 强调色 |
|------|------|------|--------|
| **北极星** | #0a0a0f | #1a1a2e | #00d4ff |
| **朱砂红** | #8b0000 | #1a0a0a | #ff6b6b |
| **琥珀金** | #1a1400 | #2d2006 | #ffd700 |
| **玉石绿** | #0a1a0a | #0f2d1f | #00ff88 |

### 🔤 字体选择

| 类型 | 字体 | 用途 |
|------|------|------|
| 标题 | Noto Serif SC / Playfair Display | 大气磅礴 |
| 正文 | Source Han Sans / Inter | 清晰易读 |
| 代码 | JetBrains Mono / Fira Code | 技术感 |

### ✨ 动效设计

```css
/* 丝绸般的过渡 */
transition: all 0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94);

/* 星空闪烁 */
@keyframes star-twinkle {
  0%, 100% { opacity: 0.3; }
  50% { opacity: 1; }
}

/* 璇玑旋转 */
@keyframes xuanji-spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```

### 📐 空间构图

```
打破对称     → 斜线构图、黄金比例
负空间     → 大量留白、呼吸感
层次叠加   → 前后遮挡、对角线
密度控制   → 疏可走马、密不透风
```

## 设计流程

1. **理解需求** → 明确用户、场景、目的
2. **确定风格** → 大胆选择美学方向
3. **执行细节** → 字体/配色/动效/布局
4. **微调打磨** → 每一个像素的精致
