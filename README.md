## 四大核心模块

### 1. Agent服务层 (Agent Service)
- 多模型并行推理
- 动态负载均衡
- 熔断降级保护

### 2. Rollout收集器 (Rollout Collector)  
- 多轮对话轨迹捕获
- 轨迹质量分类
- 自动去重与压缩

### 3. 璇玑PRM评估器 (XuanJi PRM)
- **Process Reward Model** - 过程奖励模型
- 多维度打分机制
-  majority voting容错

### 4. 进化引擎 (Evolution Engine)
- GRPO优势估计
- PPO裁剪损失
- LoRA高效微调
